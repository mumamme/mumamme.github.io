我的首个觉得贼有用的chrome插件

2020-08-21 工具总结 chrome, chrome extension, plugin Comments Word Count: 554(words) Read Count: 1(minutes)

之前一直有听说chrome插件，瞅了别人说的油猴插件，并没有给我很惊艳的感觉。或许是我没有这方面的需求吧。最近基于我的需要，我被别人安利了一个chrome插件叫做Google访问助手插件。通过这个插件，我可以访问Google相关网站了，顺便还申请了gmail（以前好像有，但是早忘了）。
贴一下他的链接：Chrome 咕咕牛懒器(guge niú lǎn qì) 安装 及助手插件操作。
下面是我的总结。

下载安装包

我们需要准备两个包：chrome浏览器安装包和助手插件包。
我的安装包下载参考链接：
助手插件包下载参考链接：https://chrome.zzzmh.cn/info?token=gocklaboggjfkolaknpbhddbaopcepfp
网盘链接：

安装chrome浏览器

直接运行chrome浏览器安装包就行。不够我好像没有发现可以自定义安装目录的地方，如果有知道的可以分享一下。

安装插件

在浏览器中输入chrome://extensions/，打开右上角的开发者模式，将.crx文件直接拖进来安装插件。

  • 如果右上角有弹窗，不需要开启同步。
  • 如果发现页面提示试用时间，不用担心，直接点击右上角，选择最下面的退出选项就可以。

再次打开chrome浏览器，就会发现可以使用Google相关服务啦。

可能的问题

如果出现了CRX_HEADER_INVALID报错(我从极简插件上下载的就会出现)，可以修改.crx文件的拓展名改成.zip，再拖进来就没有问题了，亲测有效。

整个安装过程还是比较方便的，小白友好。
不过这玩意儿不等于翻墙，所以其他很多被墙的还是不能上。在写这篇博文的时候，中美关系非常不好。前段时间比较热门的话题就是特朗普因为tiktok宣布封禁了tiktok和微信。希望后面选举结束后，状况能够改善。
