小白搭建hexo博客总结

2020-08-02 工具入门 GitHub, bolg, hexo Comments Word Count: 1k(words) Read Count: 3(minutes)

时隔半年，我终于想起我搭的这个博客了。虽然网上很多类似的教程或者经验博客了，但是我还是打算记录下，也算是自己动手的笔记了。
小白推荐视频把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo，非常详细，非常基础，跟着这个视频一步步做下来，基本上都能成功。
本文大都参考别处，侵删。

部署

本地生成博客

  1. 创建blog文件夹；
  2. 初始化博客；
  3. 启动博客；
  4. 在浏览器中输入localhost:4000，就可以看到自己的博客啦。hexo默认会创建一篇博文Hello World，记录了快速使用hexo的几个命令。

新建一篇博文

  1. 使用Ctrl+C退出服务；
  2. 输入命令hexo n "[bolg_name]"，可以看到系统自动创建了一个md文件；
  3. 进入文件目录；
  4. 基于markdown语法修改博文，保存；
  5. 生成博文；
  6. 输入hexo s重新启动博客，就可以看到刚刚创建的博文了。

创建GitHub仓库

  1. 注册GitHub账号并登录；
  2. 在右上角New repository；
  3. **Repository name一定设置成[个人GitHub账号名].github.io.**例如，我的名字是mumamme，我填的就是mumamme.github.io。

安装部署插件

  1. 在blog文件夹下安装插件；
  2. 如果看到warning，不需要关注。

到这里博客就算基本搭建成功啦，后面就是优化了。

更换主题

hexo博客的主题非常多，虽然看到了很多大佬的主题推荐，但是作为一个女生，还是更喜欢小众且颜值高的皮肤。鉴于自己水平有限，先去Themes|Hexo上挑选自己心仪的皮肤，以后再根据自己的想法修改。很多主题是需要额外安装插件的，所以需要查看作者写的文档，否则就会报错，对小白不友好。如果觉得麻烦或者选择困难，可以直接去网上搜hexo的主题推荐。
